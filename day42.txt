Handling the  Cascading of  failure  through  FALL BACK  method  using
Cloud  services  Cercuit Breaker METHOD
For this you need to add  following depenedency
spring-cloud-services-starter-cercuit-breaker

For this  depenedency u need to eneble  resilence feature
it  is  a  feature  which eliminates  cascading failure and
will not effect  other micro services if the  error occurs and  handles
the error
   
Hystrix  Cercuit breaker is removed from latest  spring  cloud  version

add  following things  in pom

<spring-cloud-services.version>2.4.1</spring-cloud-services.version>


			
			<dependency>
				<groupId>io.pivotal.spring.cloud</groupId>
				<artifactId>spring-cloud-services-dependencies</artifactId>
				<version>${spring-cloud-services.version}</version>
				<type>pom</type>
				<scope>import</scope>
			</dependency>
			


		<dependency>
		<groupId>io.pivotal.spring.cloud</groupId>
				<artifactId>spring-cloud-services-starter-circuit-breaker</artifactId>
		</dependency>
		<dependency>
		<groupId>org.springframework.cloud</groupId>
		<artifactId>spring-cloud-starter-circuitbreaker-reactor-resilience4j</artifactId>
		
		</dependency>
