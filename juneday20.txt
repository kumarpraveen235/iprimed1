types of selectors
element  selector   
selector name  must be valid  html tag name
only one  element  name is specified
multiple selector

selector name  must be valid  html tag name
more than one  element  name is specified saperating comma
class selector
selector name must  be  user defined name  preffixed by .
and   used in class attribute  of html element
id selector

selector name must  be  user defined name  preffixed by #
and   used in id attribute  of html element
child selector
selector name  must be valid  html tag name
more than one  element  name is specified with out saperating comma
